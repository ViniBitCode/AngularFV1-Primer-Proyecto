# Java APP N6! - Repaso de Frontend + Login 
💻 | Amigos de Github, hem os dado el siguiente paso haciendo uso de Angular + Springboot para realizar un Login (un tanto mediocre).  
💻 | Siendo parte del primer proyecto fullstack, van a notar que lo que es diseño no es mi fuerte al hacerlo rápido... y al Login convengamos le faltan bastantes comprobaciones de seguridad 😅   

### **Algunas otras cosas que me faltaron poner en el sistema:**
- Para el Login: que cumpla con los requisitos de la clave, que ambas claves sean iguales, que no se pueda poner 2 veces un mismo email.
- Para el back: Haber hecho del patron DTO como corresponde para la clase Usuario.

### **Tecnicas/Tecnologias usadas para el proyecto que considero interesantes:**
- Backend: Java 21 con Spring Boot 4.0.2 | Spring Data JPA + MySQL 9.5.0 | Uso de Maven para gestionar las dependencias.
- Fronted: Angular 21 + Angular material (para los campos del Register y Login)
